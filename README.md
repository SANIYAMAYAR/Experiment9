# Experiment9

Aim
To create a simple Android Widget Application that displays text on the home screen and updates it automatically.

Algorithm 
Start the project in Android Studio.
Create Activity_main.xml for basic UI.
Design Widget_layout.xml with a TextView.
Create Widget_info.xml to define widget size and update time.
Create a class extending AppWidgetProvider.
Override onUpdate() method.
Use RemoteViews to control widget layout.
Set text using setTextViewText().
Update widget using AppWidgetManager.
Register widget in AndroidManifest.xml.
Run the app and add widget to home screen.
